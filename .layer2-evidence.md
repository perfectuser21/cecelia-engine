### C1: 行号方法提取证据块
```bash
$ grep -A5 "找到下一个" scripts/release-check.sh
            # 找到下一个 ### 的行号（作为结束）
            # 注意：grep 无匹配时返回 exit 1，需要 || true 防止 set -e 退出
            END_LINE=$(tail -n +$((START_LINE + 1)) "$L2_EVIDENCE_FILE" 2>/dev/null | grep -n "^### " | head -1 | cut -d: -f1 || true)

HTTP_STATUS: 200
```
