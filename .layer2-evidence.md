## 说明

本次 release 的核心改动是将 Evidence 生成移到 CI 环境：
- **旧模式**：本地生成 Evidence → 提交到 git → SHA drift 问题
- **新模式**：CI 生成 Evidence → 不提交 git → SSOT（Single Source of Truth）

因此，本 release 没有传统的截图证据，而是通过自动化测试和脚本验证。
