# PRD - 修复 head -n -1 兼容性问题

Priority: P1

## 背景

release-check.sh 使用 `head -n -1` 在 CI 环境（Ubuntu）中可能有兼容性问题，导致 L2B 检查在 C3/C4 时失败。

## 目标

使用 `sed '$d'` 替代 `head -n -1`，提高跨平台兼容性。

## 验收标准

- CI 环境 release-check 通过
- C5-001 回归测试通过
