# PRD - Release v11.2.11

## 概述

里程碑发布：v11.2.5 → v11.2.11 的所有变更。

## 变更内容

### Phase 1: 并发竞态修复
修复多个并发写入时的竞态条件。

### Phase 2: 跨仓库 worktree 兼容
增强跨仓库和 git worktree 环境的兼容性。

### Phase 3: Promise 信号统一
统一 completion promise 格式为 `<promise>DONE</promise>`。

### Phase 4: 文档清理
归档过期文档到 `docs/.archive/`。

### Phase 5: 关键清理
- 删除过时的 `contracts/` 目录
- 废弃 H7/W5 功能

### Phase 6: rm -rf 安全验证
添加 rm -rf 操作的安全检查。

### Phase 7: 测试覆盖
新增 19 个测试用例，提升测试覆盖率。

### Phase 8: CI 工作流清理
清理 CI 配置中的过时引用。
