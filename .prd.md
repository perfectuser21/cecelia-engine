# PRD - 清理 develop 上的遗留 PRD/DoD 文件

## 背景

PR #265 错误地将功能分支的 PRD/DoD 文件合并到了 develop 分支，违反了"PRD/DoD 只在功能分支"的规范。

## 功能描述

删除 develop 分支上不应该存在的 4 个文件：
- `.prd.md`（从 PR #265）
- `.dod.md`（从 PR #265）
- `.quality-evidence.json`（从 PR #265）
- `.quality-gate-passed`（从 PR #265）

## 成功标准

- develop 分支上不再有这 4 个遗留文件
- 不影响其他任何代码或功能
- CI 全部通过

## 非功能需求

- 无代码变更，仅删除文件
- 保持代码库整洁

## 范围

仅删除遗留文件，不涉及任何代码修改。
